# Veo 3.1 Prompt Engineering Guide

Comprehensive guide for crafting effective Veo 3.1 prompts for marketing video generation.

## Prompt Structure

```
[SUBJECT] + [ACTION] + [SETTING] + [STYLE] + [CAMERA] + [LIGHTING] + [MOTION] + [AUDIO] + [CONSTRAINTS]
```

### Components

| Component | Description | Example |
|-----------|-------------|---------|
| Subject | Main focus | "A young professional in business casual" |
| Action | What happens | "types on laptop, then looks up smiling" |
| Setting | Environment | "modern co-working space with plants" |
| Style | Visual treatment | "clean, minimalist, corporate aesthetic" |
| Camera | Shot type/movement | "medium close-up, slight dolly in" |
| Lighting | Light setup | "soft natural window light" |
| Motion | Movement quality | "smooth, steady, professional" |
| Audio | Sound elements | "ambient office sounds, soft typing" |
| Constraints | Technical specs | "8 seconds, 1080p, 16:9" |

## Audio Directives

### Dialogue Format

```
"{Speaker name}" says: "{Exact quote here}"
```

**Rules:**
- Speaker name in quotes
- Dialogue in separate quotes
- Keep under 20 words per 8 seconds
- Clear, natural pacing

**Examples:**
```
"Sarah" says: "This changed everything for our team"
"Narrator" says: "Introducing the future of productivity"
```

### Sound Effects (SFX)

```
SFX: {description} as {action_trigger}
```

**Examples:**
```
SFX: Satisfying click as button is pressed
SFX: Whoosh sound as product slides into frame
SFX: Notification ping when message arrives
SFX: Soft pop as feature appears
```

### Ambience

```
Ambient: {2-3 elements max}
```

**Examples:**
```
Ambient: Busy coffee shop murmur, espresso machine
Ambient: Office typing, distant phone rings
Ambient: Wind through trees, birdsong
```

### Music

```
Music: {genre/mood}, subtle
```

**Examples:**
```
Music: Upbeat corporate, subtle background
Music: Inspiring orchestral, building
Music: Lo-fi hip hop, relaxed
Music: Electronic pop, energetic
```

## Camera Movements

| Movement | When to Use |
|----------|-------------|
| Static | Testimonials, talking head |
| Dolly in | Building emotion, focus |
| Dolly out | Reveal, context |
| Pan | Following action, scanning |
| Tilt | Vertical reveal |
| Orbit | Product showcase |
| Tracking | Following subject |

**Format:**
```
Camera: {shot_type}, {movement}, {speed}
```

## Shot Types

| Shot | Best For |
|------|----------|
| Extreme close-up | Details, emotion |
| Close-up | Products, faces |
| Medium | Conversations, demos |
| Wide | Environment, context |
| Aerial | Scale, establishing |

## Timing Control

### Explicit Timing

```
0-2s: [Scene setup and hook]
2-4s: [Main action develops]
4-6s: [Key moment/reveal]
6-8s: [Resolution and CTA]
```

### Pacing Descriptors

- "Slow, contemplative pace"
- "Quick cuts, energetic"
- "Steady, professional rhythm"
- "Building momentum"

## Common Mistakes to Avoid

### Prompt Issues

| Don't | Do |
|-------|---|
| Vague subjects | Specific descriptions |
| Multiple complex actions | Single clear action |
| Conflicting styles | Consistent aesthetic |
| Technical jargon | Natural language |
| Too much dialogue | Brief, impactful speech |

### Audio Mistakes

- More than 3 ambient elements
- Dialogue over 20 words
- Conflicting audio layers
- Unrealistic sound expectations

## Iteration Strategy

### If Quality Issues

1. Simplify prompt (fewer elements)
2. Make subject more specific
3. Reduce audio complexity
4. Use established camera moves

### If Audio Sync Issues

1. Shorten dialogue
2. Reduce simultaneous audio layers
3. Add explicit timing markers
4. Fall back to music-only track

### If Style Inconsistent

1. Add more style descriptors
2. Reference specific aesthetics
3. Use consistent lighting terms
4. Maintain single mood throughout
