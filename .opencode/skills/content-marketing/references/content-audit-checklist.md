# Content Audit Checklist

## Audit Dimensions

### Traffic Performance
- [ ] Organic sessions
- [ ] Total pageviews
- [ ] Traffic trend (up/down/flat)
- [ ] Traffic sources

### Engagement Metrics
- [ ] Time on page
- [ ] Bounce rate
- [ ] Scroll depth
- [ ] Social shares

### Conversion Performance
- [ ] Goal completions
- [ ] Conversion rate
- [ ] Lead quality (if applicable)

### SEO Health
- [ ] Target keyword ranking
- [ ] Backlink count
- [ ] Internal links
- [ ] Page speed score

### Content Quality
- [ ] Accuracy (still correct?)
- [ ] Freshness (outdated?)
- [ ] Relevance (still needed?)
- [ ] Brand alignment

## Audit Actions

| Action | Criteria |
|--------|----------|
| Keep | High traffic, converting, evergreen |
| Update | Good traffic, outdated info |
| Consolidate | Multiple thin posts on same topic |
| Redirect | Low traffic, better content exists |
| Delete | No traffic, no value, can't fix |

## Audit Report Template

```markdown
## Content Audit: [Site/Section]

### Summary
- Total pages audited: [N]
- Keep: [N]
- Update: [N]
- Consolidate: [N]
- Redirect: [N]
- Delete: [N]

### Priority Updates
| URL | Issue | Action | Priority |
|-----|-------|--------|----------|

### Quick Wins
[Low effort, high impact items]

### Recommendations
1. [Action item]
2. [Action item]
```

## Quarterly Review Cadence

1. Export content inventory
2. Pull performance data
3. Score each piece
4. Categorize actions
5. Prioritize updates
6. Execute and track
