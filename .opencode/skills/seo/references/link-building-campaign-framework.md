# Link Building Campaign Framework

## Campaign Types

### 1. Guest Post Outreach
**Best For**: Building authority, brand awareness
**Target DA**: 40+
**Success Rate**: 10-15%
**Timeline**: 2-4 weeks per placement

**Process**:
1. Identify blogs accepting guest posts
2. Analyze content quality and audience
3. Pitch relevant topic ideas
4. Write high-quality content (1500+ words)
5. Include contextual link (1-2 per post)

### 2. Resource Page Link Building
**Best For**: Quick wins, educational content
**Target DA**: 30+
**Success Rate**: 15-20%
**Timeline**: 1-2 weeks

**Process**:
1. Find resource pages (search: "intitle:resources" + keyword)
2. Create valuable resource (tool, guide, infographic)
3. Pitch addition to existing lists
4. Highlight unique value proposition

### 3. Broken Link Building
**Best For**: Replacing dead competitor links
**Target DA**: 35+
**Success Rate**: 20-30%
**Timeline**: 2-3 weeks

**Process**:
1. Find broken links on target sites (Check My Links extension)
2. Create replacement content
3. Notify site owner of broken link
4. Suggest your content as replacement

### 4. HARO (Help A Reporter Out)
**Best For**: High-authority press links
**Target DA**: 60+
**Success Rate**: 5-10%
**Timeline**: Ongoing daily

**Process**:
1. Subscribe to HARO emails (3x daily)
2. Respond to relevant queries within 2 hours
3. Provide expert quotes with credentials
4. Follow up if featured

### 5. Digital PR
**Best For**: Brand mentions, high-DA links
**Target DA**: 70+
**Success Rate**: 5-8%
**Timeline**: 4-8 weeks

**Process**:
1. Create newsworthy content (data study, survey, report)
2. Build journalist outreach list
3. Craft press release
4. Pitch to relevant journalists
5. Follow up strategically

### 6. Niche Directory Submissions
**Best For**: Local SEO, niche authority
**Target DA**: 25+
**Success Rate**: 80-90%
**Timeline**: 1 week

**Process**:
1. Identify quality directories (not spam farms)
2. Submit accurate business info
3. Track submission status
4. Monitor for approval

## Target Selection Criteria

### Domain Metrics
- **Minimum DA/DR**: 30 (adjust by campaign type)
- **Organic Traffic**: 5k+/month
- **Spam Score**: <30%
- **Active Content**: Published within 30 days

### Relevance Criteria
- **Topical Alignment**: Same/adjacent niche
- **Audience Overlap**: Target demographics match
- **Content Quality**: Well-written, valuable
- **Engagement**: Comments, shares, backlinks

### Link Opportunity Signals
- "Write for us" page exists
- Resource/links page exists
- Competitor backlinks present
- Active blog with guest authors
- Contact info easily accessible

## Outreach Workflow

### Phase 1: Research (Week 1)
- Identify 100 target prospects
- Score by DA, relevance, opportunity
- Export to outreach spreadsheet
- Find decision maker contact info

### Phase 2: Initial Outreach (Week 2)
- Send personalized emails (20-30/day)
- A/B test subject lines
- Track open/response rates
- Log all communications

### Phase 3: Follow-Up (Week 3-4)
- First follow-up: 3 days after initial
- Second follow-up: 7 days after first
- Final follow-up: 14 days after second
- Mark unresponsive as "dead"

### Phase 4: Execution (Week 5-8)
- Fulfill content requirements
- Submit for review
- Make revisions if needed
- Confirm publication + link

### Phase 5: Tracking (Ongoing)
- Monitor link status monthly
- Track referral traffic
- Measure ranking impact
- Build relationship for future opportunities

## Follow-Up Sequences

### Guest Post Sequence
**Email 1 (Day 0)**: Personalized pitch with topic ideas
**Email 2 (Day 3)**: "Just checking in" + additional value
**Email 3 (Day 10)**: Alternative angle/topic offer
**Email 4 (Day 21)**: Final attempt with social proof

### Resource Link Sequence
**Email 1 (Day 0)**: Notify of resource + value prop
**Email 2 (Day 5)**: Brief reminder with quick link
**Email 3 (Day 14)**: Final gentle nudge

### Broken Link Sequence
**Email 1 (Day 0)**: Helpful broken link notification
**Email 2 (Day 4)**: Offer replacement content
**Email 3 (Day 12)**: Last follow-up

## Success Metrics

### Campaign KPIs
- **Response Rate**: Target 30%+
- **Conversion Rate**: Target 10%+ (link secured)
- **Average DA**: Target 40+
- **DoFollow Ratio**: Target 70%+
- **Time to Link**: Target <30 days

### ROI Metrics
- **Cost per Link**: Track time/money invested
- **Link Value**: DA × Relevance × Placement score
- **Ranking Impact**: Target keyword position changes
- **Referral Traffic**: Clicks from acquired links
- **Domain Authority Growth**: Monthly DR/DA tracking

### Quality Metrics
- **Link Retention**: >95% after 6 months
- **Toxic Score**: <5% of acquired links
- **Editorial Ratio**: 80%+ editorial (not footer/sidebar)
- **Anchor Text Diversity**: <10% exact match

## Campaign Optimization

### A/B Testing Elements
- Subject lines (test 2-3 variants)
- Email length (short vs detailed)
- Call-to-action wording
- Personalization depth
- Send day/time

### Performance Analysis
- Track response rates by template
- Identify high-converting topics
- Monitor DA correlation with success
- Analyze follow-up effectiveness
- Measure seasonal trends

### Continuous Improvement
- Update templates based on data
- Refine targeting criteria
- Adjust outreach volume
- Optimize follow-up timing
- Build relationship database
