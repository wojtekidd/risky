# Ad Copy Templates

## Google Search Ads

### Headline Formulas (30 chars max)
```
[Keyword] - [Benefit]
[Action] [Product] Today
[Number]% Off [Product]
Free [Offer] | [Brand]
[Solution] for [Audience]
```

### Description Formulas (90 chars max)
```
[Benefit]. [Feature]. [Social proof]. [CTA].
Get [result] in [timeframe]. [Differentiator]. Try free.
[Problem]? [Solution]. [Proof point]. Shop now.
```

### Example
```
H1: Marketing Automation Tools
H2: Save 10+ Hours Per Week
H3: Free 14-Day Trial
D1: Automate your marketing campaigns. Used by 10,000+ businesses. Start free today.
D2: All-in-one platform for email, social, and ads. No credit card required. Sign up now.
```

## Meta/Facebook Ads

### Primary Text (125 chars visible)
```
[Hook - pain point or desire]
[Solution - what you offer]
[Proof - social proof or result]
[CTA - what to do next]
```

### Headline (40 chars)
```
[Benefit] + [Urgency/Specificity]
```

### Example
```
Primary: Tired of wasting money on ads that don't convert?

Our AI-powered platform has helped 500+ businesses cut ad spend by 40% while doubling conversions.

Click below to get your free ad audit.

Headline: Free Ad Audit - Limited Spots
CTA: Learn More
```

## LinkedIn Ads

### Intro Text (150 chars recommended)
```
[Industry insight or stat]
[Your solution]
[CTA]
```

### Example
```
78% of B2B buyers start with a Google search.

Our SEO platform helps you rank #1 for high-intent keywords.

Download our free guide to see how.
```

## Copy Best Practices

1. **Lead with benefit**, not feature
2. **Use numbers** - specific > vague
3. **Include social proof** when possible
4. **Match ad to landing page** copy
5. **Test 3-5 variations** per ad group
6. **Front-load keywords** for search
