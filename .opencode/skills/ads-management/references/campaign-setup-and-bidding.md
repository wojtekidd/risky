# Campaign Setup & Bidding Strategies

## Google Ads Campaign Types

| Type | Best For | Min Data |
|------|----------|----------|
| Search | High-intent, bottom-funnel | Keyword list + negatives |
| Display | Retargeting, awareness | Creative assets |
| Video | YouTube reach | 15-30s video |
| Performance Max | Full-funnel, e-com/lead gen | 30+ conv/mo, product feed |
| Demand Gen | Top-funnel social-style | Strong video/image creative |

### Search Campaign Setup
- Use Responsive Search Ads (RSA): 15 headlines, 4 descriptions min
- Tight ad groups: 3-5 closely related keywords per group
- Enable all assets: sitelinks, callouts, structured snippets, images
- Match types: Exact + Phrase for control; Broad only with Smart Bidding + data

### Performance Max Setup
- Needs 30+ conversions/month (50+ ideal)
- Add campaign-level negative keywords (up to 10,000)
- Provide diverse assets: headlines, descriptions, images, logos, videos
- Create intentional video (auto-generated performs poorly)
- Separate campaigns by product category/margin tier
- Audience signals: Customer Match + remarketing lists
- Learning period: 2-4 weeks; wait for 50 conversions before changes

### Demand Gen Setup
- Separate prospecting vs remarketing campaigns
- Tailor creative per placement: short videos for Shorts, carousels for Discover
- Launch with Maximize Conversions → switch to tCPA after 50+ conversions
- Allow 60-90 days to stabilize

## Meta Ads Campaign Structure

### Recommended: 2-Campaign System
- **Campaign A: Creative Testing** — small budget, 3-5 ad sets, ABO
- **Campaign B: Scaling Winners** — CBO, broad targeting, top creatives only

### Advantage+ Campaigns
- Best for e-commerce with catalog, $5K+/mo spend
- Set existing customer budget cap (e.g., 20%)
- Requires: trained Pixel, product catalog, CAPI
- Expected: 12-32% ROAS improvement vs manual

### Key Rules
- Weekly budget ≥ 50× target CPA to exit learning phase
- 72-hour rule: don't touch campaigns for 3 days post-launch
- Broad targeting outperforms narrow interest stacking for $5K+/mo accounts

## Bidding Strategies

### Google Ads
| Strategy | When to Use | Min Data |
|----------|-------------|----------|
| Maximize Clicks | New campaigns, no conversion data | None |
| Manual CPC + eCPC | Testing phase, low volume | None |
| Maximize Conversions | Have conversion tracking | 15-20 conv/mo |
| Target CPA | Stable lead gen | 30+ conv/mo |
| Target ROAS | E-com with values | 50+ conv/mo |

**Rules:**
- Don't set tCPA/tROAS below actual average
- Change budget and bid targets separately
- Allow 4-week stabilization after bid strategy change

### Meta Ads
- **CBO**: preferred for prospecting (algorithm distributes budget)
- **ABO**: for retargeting (manual control per ad set)
- Vertical scaling: increase budget max 20%/day
- Horizontal scaling: duplicate winning ad sets with new audiences
- Scaling threshold: ROAS > target for 3+ consecutive days

## Budget Management

- Google: daily budget × 30.4 = monthly ceiling (can overspend 2× daily)
- Meta: start $20-50/day per ad set, prove CPA, then scale
- Reserve 20% of budget for testing
- Retargeting always-on: even $5-10/day pays back
- 60/40 (Google/Meta) common starting split for e-commerce

## Quality Score (Google)

Three pillars: Expected CTR + Ad Relevance + Landing Page Experience
- Match keyword → ad copy → landing page theme
- Mobile page speed critical: 1s delay = ~20% fewer conversions
- Assets improve CTR → improves Expected CTR component
- Target QS 7+ on core keywords; 9-10 unlocks lowest CPCs
