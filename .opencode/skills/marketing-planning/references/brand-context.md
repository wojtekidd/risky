# Brand & Context Understanding Phase

**When to skip:** If provided with brand context or existing strategy docs, skip this phase.

## Core Activities

### Brand Context Extraction
- Run `inject-brand-context.cjs` to extract brand guidelines
- Understand brand voice, tone, and personality
- Review visual identity (colors, typography, imagery)

### Essential Documentation Review
ALWAYS read these files first:

1. **`./docs/brand-guidelines.md`** (IMPORTANT)
   - Brand voice and tone
   - Visual identity standards
   - Messaging frameworks
   - Content pillars

2. **`./docs/marketing-overview.md`**
   - Marketing strategy and positioning
   - Target audience definitions
   - Campaign history and learnings

3. **`./docs/project-overview-pdr.md`**
   - Business context and goals
   - Product/service offerings
   - Value propositions

### Existing Campaign Analysis
- Review past campaign performance
- Identify successful patterns and learnings
- Note failed approaches to avoid
- Understand channel mix and ROI

### Asset Inventory
- Review existing marketing assets
- Identify reusable content and templates
- Note gaps in asset library
- Understand asset creation capabilities

### Integration Planning
- Identify how campaigns integrate with sales process
- Map marketing automation and CRM touchpoints
- Understand attribution and tracking setup
- Consider cross-channel coordination

## Best Practices

- Start with brand docs before campaign planning
- Document learnings for consistency
- Note any brand inconsistencies to address
- Consider impact on existing initiatives
