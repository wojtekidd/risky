# YouTube Thumbnail Styles Reference

## 15 Art Direction Styles

### 1. Facecam
- **Layout:** Face on left third (cropped shoulders-up), text on right
- **Background:** Solid color or gradient matching brand
- **Face:** Expressive emotion — surprise, excitement, disbelief
- **Text:** 1-3 bold words, 80px+ with stroke outline
- **Best for:** Vlogs, reactions, reviews, commentary
- **CTR tip:** Exaggerated expressions outperform neutral by 2-3x

### 2. Before-After
- **Layout:** Split vertically — "before" left, "after" right
- **Divider:** Arrow, lightning bolt, or hard color split
- **Contrast:** Make the difference dramatic and obvious
- **Text:** "BEFORE" / "AFTER" or result metric
- **Best for:** Tutorials, transformations, fitness, makeovers
- **CTR tip:** The bigger the contrast, the higher the click

### 3. Listicle
- **Layout:** Large number (200px+) as hero element
- **Background:** Clean gradient or solid with subtle grid
- **Grid:** Small preview images of items in the list
- **Text:** "TOP [N]" or just the number
- **Best for:** Top-N videos, rankings, compilations
- **CTR tip:** Odd numbers (7, 11, 13) outperform even

### 4. Tutorial
- **Layout:** Clean instructional composition
- **Elements:** Tool/subject centered, step preview, result teaser
- **Background:** Light (white/cream) or dark (charcoal)
- **Text:** Action verb — "BUILD", "CREATE", "LEARN"
- **Best for:** How-to, educational, DIY, coding
- **CTR tip:** Show the end result, not the process

### 5. Reaction
- **Layout:** Face (exaggerated) left, subject/screenshot right
- **Expression:** Jaw-drop, wide eyes, hands on face
- **Background:** Bright accent color or subject screenshot
- **Text:** "OMG", "WOW", "NO WAY" — emotional trigger
- **Best for:** Commentary, reactions, drama, news
- **CTR tip:** The more extreme the expression, the more clicks

### 6. Cinematic
- **Layout:** Full-bleed hero image, widescreen feel
- **Lighting:** Golden hour, dramatic shadows, film-like
- **Color grading:** Teal-orange, moody blues, warm golds
- **Text:** Minimal or none — let the image speak
- **Best for:** Travel, documentary, film, storytelling
- **CTR tip:** Horizon/landscape shots with human figure for scale

### 7. Bold Text
- **Layout:** Text fills 60%+ of the frame
- **Background:** Simple solid or gradient
- **Typography:** Oversized, bold condensed, high contrast
- **Imagery:** Minimal — small icon or face as accent
- **Best for:** Opinion, hot takes, news, announcements
- **CTR tip:** Controversial or strong statements drive curiosity

### 8. Mystery / Curiosity
- **Layout:** Subject partially hidden, blurred, or circled
- **Effects:** Gaussian blur, red circle, red arrow, "?"
- **Background:** Dark or obscured scene
- **Text:** Question or teaser — never the answer
- **Best for:** Reveals, mysteries, storytelling, secrets
- **CTR tip:** Highest CTR style when done right — never give away the answer

### 9. Product Showcase
- **Layout:** Product hero shot centered on clean bg
- **Background:** Gradient (dark-to-light) or solid brand color
- **Angle:** 3/4 view, slight tilt, dramatic shadow
- **Text:** Product name + verdict ("BEST", "WORTH IT?")
- **Best for:** Reviews, unboxing, tech, gadgets
- **CTR tip:** Show the product at its best angle, not the box

### 10. Comparison (VS)
- **Layout:** Split screen, two subjects side by side
- **Center:** "VS" badge, lightning bolt, or divider
- **Colors:** Each side gets its own color zone
- **Text:** Subject names on each side
- **Best for:** Comparison videos, battles, reviews
- **CTR tip:** Make both options look appealing to create tension

### 11. Meme / Trending
- **Layout:** Follows recognizable meme template
- **Style:** Internet-native, casual, relatable
- **Typography:** Impact font or meme-standard fonts
- **Tone:** Humorous, self-aware, culturally relevant
- **Best for:** Entertainment, comedy, trending topics
- **CTR tip:** Use currently trending formats, refresh frequently

### 12. Minimalist
- **Layout:** 1-2 elements max, lots of whitespace
- **Background:** Single color or very subtle gradient
- **Typography:** Thin modern sans-serif, precise spacing
- **Elements:** One strong focal point
- **Best for:** Tech, SaaS, professional, premium content
- **CTR tip:** Stands out by being different from busy thumbnails

### 13. Dark Dramatic
- **Layout:** Dark background with glowing subject
- **Lighting:** Rim light, neon glow, spotlight effect
- **Colors:** Dark base with neon accents (cyan, magenta, green)
- **Effects:** Light rays, particles, smoke
- **Best for:** Gaming, thriller, action, horror, tech
- **CTR tip:** Glow effects on dark backgrounds are highly eye-catching

### 14. Bright Pop
- **Layout:** Full bright background, fun elements scattered
- **Colors:** Saturated primaries — yellow, cyan, magenta, lime
- **Typography:** Rounded, bold, playful fonts
- **Elements:** Stickers, stars, emoji-like shapes
- **Best for:** Kids, lifestyle, entertainment, vlogs
- **CTR tip:** Use complementary color pairs for maximum pop

### 15. Screenshot
- **Layout:** Actual UI/app screenshot with annotations
- **Annotations:** Red circles, arrows, highlight boxes
- **Background:** Dark mode UI or browser chrome
- **Text:** Feature name or "NEW" / "HOW?"
- **Best for:** Software demos, walkthroughs, bug reveals
- **CTR tip:** Annotate the specific thing that matters, not the whole screen

### 16. Diagram
- **Layout:** Excalidraw-style hand-drawn diagram fills right two-thirds, optional face on left third
- **Diagram types:** System architecture, flowchart, mind map, process flow, data pipeline
- **Style:** Rough hand-drawn lines, rounded rectangles, simple arrows, handwritten labels
- **Text:** Bold overlaid on whiteboard area, top-left or bottom
- **Best for:** Tech explainers, system design, architecture tutorials
- **CTR tip:** Authentic sketch feel outperforms polished vector diagrams — looks "real"

### 17. Whiteboard
- **Layout:** Full whiteboard background with colorful marker-style sketches
- **Elements:** Icons, arrows, annotations in blue/red/green/black markers
- **Content:** Roadmaps, concept maps, brainstorm sessions, learning paths
- **Text:** Dark text with white stroke, corner placement
- **Best for:** Educational content, roadmaps, concept explainers, brainstorms
- **CTR tip:** Filled whiteboards feel information-dense and valuable

## YouTube-Specific Design Rules

### Readability Matrix
| View Context | Size | Must Read |
|--------------|------|-----------|
| Suggested sidebar | 168 × 94px | Title text only |
| Mobile feed | 246 × 138px | Title + face |
| Home feed desktop | 360 × 202px | All elements |
| Search results | 360 × 202px | All elements |

### Dead Zones
- **Bottom-right:** Video duration badge (dark semi-transparent)
- **Bottom-left:** "Watch Later" and "Add to Queue" on hover
- **Top-right:** Channel watermark area (if enabled)

### Color Dos and Don'ts
| Do | Don't |
|----|-------|
| High contrast pairs | YouTube red (#FF0000) as primary |
| Complementary colors | Low contrast pastel-on-pastel |
| Brand-consistent palette | More than 3 colors |
| Dark bg + bright text | Light bg + light text |

### Typography Rules
- **Maximum:** 3 words
- **Minimum font:** 80px at 1280×720
- **Weight:** Bold or Extra-Bold only
- **Outline:** 3-4px stroke in contrasting color
- **Shadow:** Drop shadow for depth on photo backgrounds
- **Font stacks:** Impact, Montserrat Black, Anton, Bebas Neue, Oswald Bold
