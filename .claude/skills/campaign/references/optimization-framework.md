# Campaign Optimization Framework

## Optimization Cycle

```
Monitor → Analyze → Hypothesize → Test → Implement → Repeat
```

## What to Optimize

### Creative
- Headlines/copy
- Images/videos
- CTA buttons
- Ad formats
- Landing pages

### Targeting
- Audiences
- Demographics
- Placements
- Devices
- Geography

### Bidding
- Bid strategy
- Bid amounts
- Dayparting
- Budget allocation

## Optimization by Metric

### Low CTR (<1%)
- Test new headlines
- Try different images
- Refine targeting
- Check ad relevance

### Low CVR (<2%)
- Improve landing page
- Strengthen offer
- Test CTA copy
- Check page speed

### High CPC
- Improve quality score
- Adjust bidding
- Test new audiences
- Refine keywords

### Low ROAS (<3:1)
- Pause worst performers
- Shift budget to winners
- Test higher-value audiences
- Optimize for value

## Optimization Schedule

| Timeframe | Action |
|-----------|--------|
| Daily | Monitor spend, pause disasters |
| 2-3 Days | Pause low performers |
| Weekly | Budget reallocation |
| Bi-weekly | Creative refresh |
| Monthly | Strategy review |

## Statistical Significance

Before making changes:
- Minimum 100 clicks per variant
- 95% confidence level
- Wait 3-7 days minimum

## Optimization Priorities

### Quick Wins (Do First)
- Pause zero-conversion ads
- Increase budget on winners
- Fix broken links
- Add negative keywords

### Medium Effort
- Test new audiences
- Create new ad variants
- Optimize landing pages
- Adjust bidding

### Strategic (Plan Carefully)
- New channel expansion
- Offer/pricing changes
- Major creative overhaul
- Targeting restructure
